
一个基于 Pygame 的视觉化间歇训练计时器

![示例界面](https://i-blog.csdnimg.cn/direct/348c909bc4004ec0b83cd68ffe5994e0.png)  

## 功能特点

- 🏃 **6 个运动项目**：显示为彩色方块，呈两行三列排列
- 🔁 **循环流程**：箭头清晰指示项目顺序（1→2→3→4→5→6→1）
- ⏱️ **计时器**：
  - 每个项目运动 `20` 秒
  - 项目之间休息 `10` 秒
  - 中央大数字显示剩余时间，并标注当前状态（运动中/休息中/已暂停）
- 📊 **统计信息**：
  - 总训练时间
  - 完成的组数（完整循环次数）
  - 已完成的项目总数
- 🎮 **键盘控制**：
  - `空格键`：开始 / 暂停
  - `R 键`：重置所有计时
- 🎨 **视觉反馈**：
  - **绿色**：当前正在进行的运动项目
  - **黄色**：休息状态，中央计时器背景变黄
  - **橙色**：休息时下一步将要进行的项目（箭头也变为高亮）
  - **紫色**：组数显示

## 安装

1. **安装 Python**（3.6 或更高版本）
2. **安装 Pygame**：
   ```bash
   pip install pygame
   ```
3. **下载脚本**：将 `main.py` 保存到本地。

## 使用方法

在终端运行：
```bash
python main.py
```

### 操作说明
- **开始 / 暂停**：按下 `空格键`
- **重置**：按下 `R 键`（重置后需再次按空格开始）
- **退出**：关闭窗口或按 `Ctrl+C`
